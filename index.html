Живой пример

Подключение виджета

Чтобы добавить на свой сайт виджет афиши Качканара, выполните эти два шага:

I. В тег <head> на странице Вашего сайта добавьте подключение widget.js:

<script src="https://widget.k4r.ru/widget.js"></script>

II. На странице Вашего сайта добавьте элемент DIV, задайте ему уникальный класс или ID и добавьте блок кода инициализации виджета.

<div class="k4r-events-widget"></div>
<script>
K4R.Events('.k4r-events-widget');
</script>

Дополнительные настройки

Метод K4R.Events принимает два параметра:

  • selector (string) — CSS селектор элемента, который будет являться контейнером для блока с афишей. В примере выше это значение .k4r-events-widget
  • options (object) — опции блока с виджетом. Объект, который может содержать поля:
    • header — отображение заголовка виджета. Возможные значения: 0 — скрыть заголовок. 1 — отображать (по умолчанию).
    • limit — ограничение на количество выводимых событий. По умолчанию — 10
    • width — ширина виджета. Строка с указанием величин (пиксели или проценты). По умолчанию — 100%
